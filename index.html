Introdução

Nós do grupo optamos por utilizar uma base de dados relacionadas a filmes

Porque filmes?

Com o passar dos anos os filmes foram evoluindo, tanto com efeitos especiais quanto em vendas, e vamos analisar isso para trazer dados consistentes de avaliação.


Utilizando a base de dados de filmes da The Movie Data Base (www.themoviedb.org/), que contém cerca de 490.300 filmes desde 1980 a 2019, que é atualizada diariamente por mais de 200.000 desenvolvedores e companhias.
